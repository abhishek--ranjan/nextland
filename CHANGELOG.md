# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-01-17

### Added
- Initial release of NextLand
- Landing page with feature overview
- Dashboard with stats and quick actions
- Content Management module (UI)
- Governance module with proposals, voting, meetings, and committees
- Facilities Management module with booking, maintenance, and visitor management
- Community module with forums, events, and member directory
- Docker support with Dockerfile and docker-compose.yml
- MIT License
- Contributing guidelines
- Security policy
- Comprehensive README with setup instructions

### Technology Stack
- Next.js 16.1.3 with TypeScript
- Tailwind CSS 4.x
- React 19.2.3
- Node.js 20.x
